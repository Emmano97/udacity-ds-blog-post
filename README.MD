## Installation <a name="installation"></a>

This code runs with Python version 3.8.* and requires some libraries, to install theses libraries you will need to execute: </br>
` pip install -r requirements.txt `

You will need to download the glassdoor job review Survey dataset and put in the same folder as the jupyter notebook. You can find the data to download [here](https://www.kaggle.com/datasets/davidgauthier/glassdoor-job-reviews). </br>

## Project Motivation <a name="motivation"></a>

This is an Udacity Nanodegree project.I was interested in using glassdoor job review Survey Data to better answer the bellow questions:</br>
1 - What are top 15 best companies based on glassdoor reviews?<br>
2 - How review vary based on if the reviewer is a current, former employee or a contractor?<br>
3 - What is the recommendation rate?
 </br>
